# Global Logistics Management System (GLMS)

## 📌 Overview
The **Global Logistics Management System (GLMS)** is a multi-project software solution designed to manage logistics operations such as shipment tracking, service coordination, and system integrations.

The solution follows a **monorepo architecture**, consisting of a main application, an API layer, and a dedicated test project.

## ⚙️ Technologies Used

- Entity Framework Core
- RESTful API architecture
- Unit Testing (xUnit / MSTest)
- Git & GitHub version control

## 🚀 Features

- Shipment and logistics management
- Modular API-based architecture
- Data validation and error handling
- Secure service separation (UI, API, Tests)
- Unit testing for core business logic
- Scalable monorepo structure
